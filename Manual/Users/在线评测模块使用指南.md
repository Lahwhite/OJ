## 3. 环境配置方式

使用 MySQL 中 root 用户执行以下语句创建数据库

关于启动 root 用户
```
mysql -u root -p
```
然后输入你自己的密码

```
CREATE DATABASE IF NOT EXISTS `myOJ`;
USE`myOJ`;

CREATE TABLE IF NOT EXISTS `Users` (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(32) NOT NULL UNIQUE,
    password VARCHAR(60) NOT NULL
);

CREATE TABLE IF NOT EXISTS `Profiles` (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(32) NOT NULL UNIQUE,
    nickname VARCHAR(64)  CHARACTER SET utf8mb4,
    signature VARCHAR(255)  CHARACTER SET utf8mb4,
    avatar VARCHAR(255)
);

select * from `myOJ`.`Users`;
select * from `myOJ`.`Profiles`;
```

也可以选择使用MySQL Workbench 8.0 CE 执行 UsersTest.sql 中语句

执行以下语句创建新用户 User
```
CREATE USER 'User'@'localhost' IDENTIFIED BY 'Management';
```

执行以下语句赋予 User 关于 myOJ 数据库的全部权限
```
GRANT ALL PRIVILEGES ON myOJ.* TO 'User'@'localhost';
FLUSH PRIVILEGES;
```

## 4. 服务启动方式

点击以下文件即可启动服务
```
OJ/Users/target/Users-0.0.1-SNAPSHOT.jar
```

访问以下页面
```
http://localhost:8081/
```

或输入
```bash
ipconfig
```

获取本机IPv4 地址后可访问（需要在同一局域网内）
```
http://xxx.xx.xx.xx:8081/
```

Users 目录下执行
```
mvn clean spring-boot:run
```

或在 Users 目录下打包后，点击 .jar 文件执行
```
mvn clean package
```

记得在 shell 中开启 redis 服务
```
redis-server
```
